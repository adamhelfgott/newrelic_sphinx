NewRelic Sphinx Plugin Uninstallation

Uninstallation of the plugin is quite simple. Since plugin itself is just statically linked executable file you can just terminate process and remove plugin file:

{% highlight bash %} pkill -9 sphinx_agent rm sphinx_agent {% endhighlight %}
